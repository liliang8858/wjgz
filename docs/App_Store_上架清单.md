## 📤 提交流程

### 1️⃣ Xcode准备
```bash
# 1. 清理项目
Product → Clean Build Folder

# 2. 修改版本号
Target → General → Version: 1.0, Build: 1

# 3. 选择发布配置
Edit Scheme → Run → Release

# 4. 选择目标设备
Any iOS Device (arm64)

# 5. 创建Archive
Product → Archive
```

### 2️⃣ 上传到App Store Connect
```bash
# 1. 在Organizer中选择Archive
# 2. 点击"Distribute App"
# 3. 选择"App Store Connect"
# 4. 选择"Upload"
# 5. 等待处理完成（10-30分钟）
```
